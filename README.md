# <LoopBreak> – Advitiya 26 Submission

## NYX

## Game Description
This game is developed using Unreal Engine 5.5 as part of Advitiya 26.
It focuses on immersive gameplay, atmospheric design, and theme-driven mechanics.
The experience emphasizes exploration, interaction, and narrative elements.

## Themes Incorporated
The given themes are incorporated in the following ways:
- Narrative: The storyline reflects the core theme through character motivation and world-building.
- Gameplay Mechanics: Player actions and progression are designed to reinforce the theme.
- Visual Design: Lighting, environment, and assets visually represent the theme.
- Audio: Background music and sound effects enhance emotional immersion aligned with the theme.

## Setup Instructions

### Requirements
- Unreal Engine 5.5
- Windows 10/11 (for Win64 build)

### Running from Source
1. Clone the GitHub repository
2. Open `<ThePine>.uproject` in Unreal Engine 5.5
3. Click Play to test or Package Project to build

- Windows Executable: Included in the final submission ZIP

## Controls
- WASD – Move
- Mouse – Camera
- Space – Jump
- Esc – Pause/Menu
